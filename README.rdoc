Local Table is an online marketplace where craftsmen can sell the furniture they make and consumers can easily find great furniture from their own towns and cities.

-A user will make a personal log-in for the site
-A user will be able to browse furniture by maker or by type.
-A user has the ability to post his/her own furniture portfolio on the site and make changes
-All goods  provide cost and contact information for its maker

My app uses:
-Html
-Css
-Ruby on Rails
-Deployed through Heroku
